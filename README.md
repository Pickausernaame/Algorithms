## Binary MSD для long long

Дан массив неотрицательных целых 64-разрядных чисел. Количество чисел не больше 1000000. Отсортировать массив методом MSD по битам (бинарный QuickSort).

### Формат ввода вывода

| Ввод  | Вывод |
| :---: | :-:   |
|  3 | 4 7 1000000   |
|  4 1000000 7    |       |
|  4 7 1000000     |       |
